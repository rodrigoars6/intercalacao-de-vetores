# Array Interleaving Python

Projeto desenvolvido em Python para leitura de dois vetores com 10 elementos cada e geração de um terceiro vetor contendo os elementos intercalados.

## 📚 Sobre o Projeto

Este programa realiza a leitura de dois vetores e cria automaticamente um terceiro vetor de 20 posições, intercalando os elementos dos vetores originais.

O objetivo do projeto é praticar manipulação de listas, estruturas de repetição e lógica de programação em Python.

## 🚀 Funcionalidades

- Leitura de dois vetores com 10 elementos
- Intercalação automática dos valores
- Criação de um terceiro vetor com 20 posições
- Exibição do vetor final

## 🧠 Conceitos Aplicados

- Listas (`list`)
- Estruturas de repetição (`for`)
- Manipulação de vetores
- Entrada e saída de dados
- Lógica de programação

## 💻 Tecnologias Utilizadas

- Python 3
